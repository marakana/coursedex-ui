Java™ Essentials

Java™ Essentials course (aka Java: The Good Parts) is designed to help you get up and running with Java in shortest possible time. It is designed for those who want to program in Java yet don't need to know absolutely everything. We've taken a detailed consideration in choosing only the most important parts of the language and the platform that you're likely going to need regardless where your Java development takes you in the future.

Overview of Java

In this module, you will get a quick high-level overview of the Java ecosystem, its history and how we got here.

  • History of Java
  • What Java is
  • Why Java
  • Java after 15+ years

Hello Java

In this module, you will write a simple Hello, World program in Java. You will dissect this program and learn about all of its major moving parts.

  • Installing and configuring Java Virtual Machine
  • Implementing HelloWorld in Java (lab)
  • Java class/file structure and naming conventions
  • Java keywords and identifiers
  • Compiling and running Java programs
  • Comments in Java code
  • The main() method
  • Installing, configuring, and using the Eclipse IDE
